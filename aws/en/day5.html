Elastic Beanstalk

Elastic Beanstalk

  • simplifies application deployment
  • quick start with AWS of your app
  • overlay of over 100 other AWS services
  • runs your code on EC2 instances
  • user chooses platform which has to run code

Elastic Beanstalk - platforms

  • Packer Builder
  • Single Container Docker
  • Multicontainer Docker
  • Preconfigured Docker
  • Go
  • Java SE
  • Java with Tomcat
  • .NET on Windows Server with IIS
  • Node.js
  • PHP
  • Python
  • Ruby

Elastic Beanstalk - Docker

  • integrates with ECS and ELB
  • no-config autoscaling
  • hard memory limits
  • Docker image is build by AWS on their side during deployment

Elastic Container Service

  • AWS container orchestration tool
  • gives you full controll
  • obraz budowany poza infrastrukturÄ…
  • hard and soft memory limits

How it works? (ECS)

  • runs on EC2 instances
  • user becomes owner of EC2 instances
  • user maintains EC2 instances
  • allows to mount external volumes (EBS/S3/IS)

Fargate (ECS)

  • user doesn't manage EC2 instances
  • user is not owner of EC2 instances
  • no abbility to mount volumes
  • more expensive than classic ECS
  • recommended for apps integrated over the network

Elastic Kubernetes Service

Elastic Kubernetes Service

  • runs Kubernetes stacks in AWS cloud
  • no changes around Kubernetes
  • claster runs on EC2 instances
  • user becomes owner of EC2 instances
  • machines run AWS Linux 2
  • you can use different AMI, with EKS agent preinstalled
  • works also with Fargate
